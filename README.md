# SolvePath

SolvePath is an educational Python mathematics library that provides clear and beginner-friendly, step-by-step solutions to a wide range of mathematical problems.

The library prioritizes:
- correctness,
- safety,
- structured explanations,
- and consistent output formatting,

making it suitable for learning, teaching, and academic use.


---

## Features

- Step-by-step mathematical solutions (teacher-style explanations)
- Beginner-friendly explanations written in a teacher-style format
- Consistent and easy-to-use API across all modules
- Defensive input validation for safer usage
- No use of `eval` or unsafe execution methods
- No automatic printing (functions always return results explicitly)
- Designed for learning, teaching, and self-study
